Profile Selection Method
Instructions for ordering:
1. Select the panel design (Roma, Milano, etc).

2. Select the profile type (R1, R2, R3, R4, F1, F2, F3, F4).

3. Select the sticking profile from this page (Ogee Profile,
Shaker Profile, Cove & Bead Profile, Contemporary Profile).

Note that with R3, R4, F3 & F4 there is no sticking profile. The
sticking is replaced with an applied moulding. For this style,
select “shaker sticking with _____ moulding”.
